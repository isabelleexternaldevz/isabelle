## 🔧 Troubleshooting & Common Issues

### ❌ Injection Failed / "Injection Error" Message

**Cause:** The game hasn't fully loaded, or a previous injection attempt left a stuck process.

**Fix:**
```bash
# Kill leftover Isabelle processes (Windows)
taskkill /F /IM IsabelleLoader.exe
# Then relaunch and wait for full game load before injecting
```

### ⚠️ Scripts Not Executing After Injection

**Cause:** Script syntax error or Luau compatibility issue.

**Fix:** Enable `verbose_log: true` in config, re-execute, and check the log panel for the exact error line. Test the script in Sandbox Mode first.

### 📱 APK Crashes on Android Launch

**Cause:** Missing permissions or incompatible Android version (below 8.0).

**Fix:** Go to **App Info → Permissions** and manually enable all permissions. If on Android 7.x, the APK is not supported — update your OS.

### 🍎 macOS "App is Damaged" Error

**Cause:** macOS Gatekeeper blocking the app.

**Fix:**
```bash
# Remove quarantine flag
xattr -cr /Applications/Isabelle.app
# Then relaunch normally
```

### 🐢 Slow Injection Speed

**Cause:** High `inject_delay_ms` value or background system load.

**Fix:** Lower `inject_delay_ms` to `800` in settings, and close unnecessary background applications.

### 🔑 iOS Certificate Revoked

**Cause:** Apple revoked the signing certificate.

**Fix:** Re-sign the IPA using your sideloading tool and re-install. This happens periodically and is expected.
